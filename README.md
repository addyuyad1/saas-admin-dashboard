## 🏗️ Project Architecture

The project follows a **feature-based scalable architecture**:

```
src/
 ├── app/
 │   ├── core/              # Singleton services, models, guards
 │   ├── shared/            # Reusable components, directives, pipes
 │   ├── features/          # Feature modules (dashboard, users, analytics)
 │   ├── layout/            # Layout components (sidebar, header)
 │   └── app.routes.ts
```

### 🧠 Key Principles:

* Separation of concerns
* Lazy-loaded modules
* Reusable UI components
* API-ready service layer

## ⚡ Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/saas-admin-dashboard.git
cd saas-admin-dashboard
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run the application

```bash
ng serve
```

---

### 4️⃣ Open in browser

```
http://localhost:4200
```
